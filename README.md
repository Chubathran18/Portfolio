## 🖥️ Preview

> A cinematic, scroll-driven portfolio with dark techy vibes — built entirely with vanilla HTML, CSS, and JavaScript. No frameworks. No dependencies. Pure performance.

---

## ✨ Features

- 🎬 **Cinematic scroll experience** — sections reveal like film scenes
- 🌀 **Animated grid background** with floating parallax orbs
- ⌨️ **Custom magnetic cursor** with lagging ring effect
- 📺 **Scanline CRT overlay** for that terminal aesthetic
- 🔤 **Glitch hover effect** on hero name
- 🎞️ **Infinite scrolling skill tapes** — two rows, opposite directions
- 📊 **Scroll progress bar** acts as a timeline
- 📱 **Fully responsive** — mobile to desktop

---

## 🛠️ Built With

| Layer | Technology |
|-------|-----------|
| Structure | HTML5 |
| Styling | CSS3 (Custom Properties, Animations, Clip-path) |
| Interactivity | Vanilla JavaScript (IntersectionObserver, rAF) |
| Fonts | Space Grotesk · JetBrains Mono · Syne |

> Zero npm packages. Zero build tools. Open `index.html` and it just works.
